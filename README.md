## Bugs
False positive recognition: `p1=(0,0), p2=(4,0), p3=(2,6), p4=(2,4)`. Triangles `(p1,p2,p3), (p1,p2,p4)` should not be valid but are.

feel free to reach out to me with any bugs you discover.
